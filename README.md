
# Calculator for ATAK

ATAK Calculator plugin used to calculate target's position

________

Kalkulator do aplikacji ATAK służący do obliczania współrzędnych celu

## Funkcjonalności

- Obliczanie współrzędnych celu
- Rysowanie pozycji celu na mapie
